# 📚 学习计划助手

一个全功能的学习管理与规划工具，帮助你高效规划学习路径、追踪技能成长、获取 AI 推荐。

## ✨ 核心功能

### 📊 仪表盘
- 学习时长统计、任务完成率、技能排行榜
- 近7天学习时长可视化图表
- 一键开始学习计时

### 📝 学习计划
- 创建学习计划，拆解任务，关联技能
- 任务计时器（自动记录学习时长并转换为技能经验）
- 任务笔记功能，记录学习心得
- 优先级与截止日期管理

### 🗓️ 日历视图
- 按日期查看任务安排
- 学习记录时间线

### 🎯 技能面板
- 150+ 技能库，覆盖编程、前端、后端、数据库、AI、DevOps、数据分析、移动开发、网络安全等领域
- 技能等级与经验系统（含突破机制）
- 技能树知识点掌握追踪
- 按分类筛选与排序

### 💼 职业路径
- 职业方向推荐
- 职业项目实践

### 📁 项目目标
- 按技能难度分级的实战项目
- 完成项目解锁技能突破

### 🤖 AI 推荐
- 智能搜索：支持关键词、别名匹配，覆盖 152 个技能的学习资源
- 搜索筛选：按分类、难度筛选结果
- 搜索排序：按匹配度、难度、学习时间排序
- 搜索历史：自动记录最近搜索
- AI 对话：本地 AI（Ollama）支持，智能学习建议
- 路线规划：自动生成学习路线图
- 一键创建计划：搜索结果直接生成学习计划，自动填充学习路径任务
- API 搜索扩展：支持 SerpAPI、Google Custom Search、自定义 API

### 🧠 AI 测验
- AI 出题：内置题库 + AI 生成双模式，覆盖 14+ 热门技能
- 5 种题型：单选题、多选题、填空题、简答题、编程题
- 智能批改：自动评分 + 详细解析
- 难度分级：按技能难度智能匹配题目
- 分类出题：根据技能类别生成领域相关题目
- 测验记录：历史测验成绩追踪

### 💾 数据管理
- 数据导出（JSON 备份）
- 数据导入（一键恢复）
- 数据重置：一键清空所有学习记录，恢复初始状态
- 本地持久化存储

## 🛠️ 技术栈

- **React** + **TypeScript**
- **Zustand** 状态管理（持久化存储）
- **Recharts** 数据可视化
- **Lucide Icons** 图标
- **Vite** 构建工具

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📁 项目结构

```
src/
├── components/          # 页面组件
│   ├── Dashboard.tsx       # 仪表盘
│   ├── PlansPage.tsx       # 计划列表
│   ├── PlanEditor.tsx      # 计划编辑器
│   ├── CalendarView.tsx    # 日历视图
│   ├── SkillDashboard.tsx # 技能面板
│   ├── CareerPage.tsx      # 职业路径
│   ├── ProjectGoals.tsx    # 项目目标
│   ├── AIRecommend.tsx     # AI推荐与搜索
│   ├── StudyReport.tsx     # 学习报告
│   ├── QuizPage.tsx        # AI测验
│   └── Modal.tsx           # 统一模态框
├── store/                # 状态管理
├── data/                 # 数据源（技能、资源、项目）
├── utils/                # 工具函数
└── App.css / api-styles.css  # 样式
```
